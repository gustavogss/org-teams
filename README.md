# Projeto org-teams :busts_in_silhouette:
- Aplicativo para Android e IOS com a finalizadade de organizar grupos de pessoas para jogar

## Tecnologias e ferramentas utilizadas :robot:
- [React Native](https://reactnative.dev/docs/environment-setup)
- [Expo Manager Workflow](https://docs.expo.dev/)
- [Styled Components](https://styled-components.com/docs)
- [React Navigation](https://reactnavigation.org/docs/getting-started/)
- [Stack Navigator](https://reactnavigation.org/docs/stack-navigator)
- [SVG Transformer](https://www.npmjs.com/package/react-native-svg-transformer)
- [Babel Module Resolver](https://github.com/tleunen/babel-plugin-module-resolver)
